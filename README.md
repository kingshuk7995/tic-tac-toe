# Tic-Tac-Toe

A simple Tic-Tac-Toe game that uses a C++ minimax engine (with alpha-beta pruning) compiled to WebAssembly using Emscripten.

### Demo  
https://tic-tac-toe-navy-mu.vercel.app/

### Features
- Perfect-play AI (minimax + alpha-beta)
- C++ core compiled to WebAssembly
- Lightweight React UI
